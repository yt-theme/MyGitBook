### 创建正则表达式
可使用new创建

```
var a = new RegExp('abc','ig')
// 第一个参数是字符串
// 第二个参数是模式修饰符
	模式修饰符
    |------|--------------|
    | 参数 |     含义     |
    |------|--------------|
    |   i  |  忽略大小写  |
    |------|--------------|
    |   g  |   全局匹配   |
    |------|--------------|
    |   m  |   多行匹配   |
```

可使用//创建 (字面量)

```
let a = /abc/
let a = /abc/ig	// 带修饰符 
```

测试字符串

```
let a = new RegExp('Box')
let str = 'box'
console.log(a.test(str)) // 测试结果false,大小写不一致



let a = new RegExp('Box', 'i')
let str = 'box'
console.log(a.test(str)) // true,添加i修饰符区分大小写


|------|----------------------------------------|
| 方法 |                  功能                  |
|------|----------------------------------------|
| test | 在字符串中测试模式匹配,返回true或false |
|------|----------------------------------------|
| exec |   在字符串中执行匹配搜索,返回结果数组  |


console.log(/Box/i.test('box'))	// true 一名话匹配
```
